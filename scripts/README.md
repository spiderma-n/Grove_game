# About
All of this is the code that I wrote for each part of the game.
# Explain each part
1.**area_2d** - ระบบของเกม เช่น เวลาเรา game over เราจะรีเซ็ตกลับไปใหม่ที่จุดเกิดที่เดิมตอนแรก<br>
2.**coin.gd** - เหรียญที่ตัวละครของเราจะมี action ด้วยคือการเก็บเหรียญเพื่อนำมาเป็นแต้ม<br>
3.**gamemanager.gd** - ระบบนับคะแนนที่เราสะสมได้ระหว่างเกมและสรุปผลหลังถึงเส้นชัย<br>
4.**killzone.gd** - ระบบที่จะกำหนดว่าเมื่อผู้เล่นตกลงไปถึงระดับไหนจะถือว่า gameover และจะถูกรีเซ็ตไปที่จุดเริ่มต้น
